HTML :

HTML est l’abréviation de « hypertext markup language » (langage de balisage hypertexte) et est un langage relativement simple utilisé pour créer des pages web. Comme il n’autorise pas les variables ou les fonctions, il n’est pas considéré comme un « langage de programmation », mais plutôt comme un « langage de balisage », c’est-à-dire un langage qui utilise des balises pour définir les éléments d’un document.

Quelques exemples de balise utiliser en HTML :

  • ‹h1› ‹/h1› pour mettre un titre
  • ‹p› ‹/p› pour mettre un paragraphe
  • ‹img› pour mettre une image
  • ‹a› ‹/a› pour mettre un lien

CSS :

Cascading Style Sheets (CSS) est un langage de programmation qui vous permet de déterminer la design des documents électroniques. À l’aide de simples instructions, présentées dans des codes sources clairs, les éléments de la page Web comme la mise en page, la couleur et la police peuvent ainsi être modulés à souhait. Grâce aux feuilles de style en cascade, la structure sémantique et le contenu du document restent totalement intacts. CSS a été lancé au milieu des années 90 et est à présent considéré comme le langage de feuilles de style standard sur le World Wide Web.

Quelques exemples de balise utiliser en CSS :

  • h1{} pour modifier un titre
  • p{} pour modifier un paragraphe
  • img{}pour modifier une image
  • a{} pour modifier un lien
