PHP Quèsaco

Définition generale

PHP signifie « Personal Home Page » (page d’accueil personnelle), mais c’est maintenant l’abréviation récursive de PHP : Préprocesseur d’hypertexte. Le code PHP peut être intégré dans le code HTML ou être utilisé en combinaison avec divers systèmes de modèles web, systèmes de gestion de contenu web et cadres web.PHP est un langage de script côté serveur. Il est utilisé pour développer des sites web statiques ou dynamiques ou des applications web. PHP signifie Hypertext Pre-processor, qui signifiait auparavant Personal Home Pages. Les scripts PHP ne peuvent être interprétés que sur un serveur sur lequel PHP est installé. Les ordinateurs clients qui accèdent aux scripts PHP ne nécessitent qu’un navigateur web.
